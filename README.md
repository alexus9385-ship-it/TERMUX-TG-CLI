TERMUX TG CLI

Консольный Telegram-клиент для терминала с поддержкой нескольких аккаунтов, пагинацией чатов и сообщений, отправкой файлов и цветным интерфейсом.

---

Возможности

· Управление аккаунтами – создание, выбор и удаление сессий.
· Список диалогов – пагинация, отображение непрочитанных и статуса онлайн/оффлайн.
· Профиль – детальная информация о пользователе или группе/канале.
· Чат – просмотр истории с пагинацией, отправка текста и файлов (по пути).
· Глобальный поиск – по username, ID или номеру телефона.
· Настройки – лимит сообщений в чате и размер страницы чатов.
· Цветной вывод – Rich + Colorama.

---

Требования

· Python 3.7+
· Установка зависимостей:
  ```bash
  pip install telethon rich colorama
  ```

---

Установка

```bash
git clone https://github.com/alexus9385-ship-it/TERMUX-TG-CLI.git
cd TERMUX-TG-CLI
pip install -r requirements.txt
```

Если requirements.txt отсутствует, установите вручную:

```bash
pip install telethon rich colorama
```

---

Настройка API

Получите api_id и api_hash на my.telegram.org.
При первом запуске программа запросит эти данные и сохранит их в config.json.

---

Запуск

```bash
python main.py
```

---

Управление

Меню аккаунтов

· Выбор существующей сессии по номеру.
· ADD – добавить новый аккаунт (ввод номера телефона).
· DEL [№] – удалить сессию.
· Q – выход.

---

Главное меню

Команда Действие
1 Список чатов
2 Глобальный поиск
3 Мой профиль
4 Настройки CLI
5 Сменить аккаунт
0 Выход

---

Список чатов

Команда Действие
N Следующая страница
B Предыдущая страница
P [номер] Профиль чата/пользователя
[номер] Войти в выбранный чат

---

В чате

Команда Действие
/P Профиль текущего чата/пользователя
/N Следующая страница (или /N [число] для перехода на конкретную)
/B Предыдущая страница
BACK Выход из чата
текст Отправить текстовое сообщение
путь/к/файлу Отправить файл (если путь существует)

---

Глобальный поиск

Введите:

· @username
· +номер_телефона
· ID_пользователя

При успешном нахождении открывается профиль.

---

Настройки

Команда Действие
1 Изменить лимит сообщений в чате (history_limit)
2 Изменить количество чатов на страницу (page_size)
0 Вернуться (настройки сохраняются)

---

Конфигурация

Файл config.json хранит:

· api_id, api_hash
· history_limit (по умолчанию 20)
· page_size (по умолчанию 10)

Настройки обновляются автоматически при выходе из меню настроек.

---

Сессии

Сессии сохраняются как .session файлы.
Имя файла служит идентификатором аккаунта.
При добавлении нового аккаунта вводится номер телефона, и создаётся сессия с этим номером в имени.

---

Пример работы

```bash
$ python main.py
# Выбор аккаунта
# Главное меню → 1
# Список чатов → ввод номера чата
# Чат: отправка сообщения, /P для профиля, BACK для выхода
```

---

Завершение

· В главном меню введите 0 для выхода.
· Принудительный выход – Ctrl+C.
